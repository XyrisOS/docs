◆ tss_flush()

void tss_flush ( )

This function is defined in flush.s (assembly) and is used to flush the TSS when we (re)load into the kernel or into userspace. The extern "C" syntax tells the compiler that our function is defined somewhere else and the "C" part tells the compiler to not string mangle our function and to instead look for a literal "tss_flush" function in assembly.
