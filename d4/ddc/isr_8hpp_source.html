15 /**
16  * All of the following values are Interrupt Request (IRQ) identifiers
17  * These values start at 32 because there are 32 prior values reserved
18  * for processor level exceptions. Look at interrupt.s for the ASM on
19  * how the IRQs call functions (and how they pass their value to said
20  * function). Each Interrupt Request pushes the IRQ value along with
21  * their corresponding hardware interrupt value (starting at 32).
22  */
